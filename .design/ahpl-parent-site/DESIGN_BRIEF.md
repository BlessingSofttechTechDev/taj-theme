# Design Brief — AHPL Parent Website

> Source of truth: `~/Downloads/AHPL_Website_Content_Response_Final.docx`
> This file is a working summary for the design-review skill — the docx is canonical.

## Project

AHPL parent site (Layer 1) with a Taj Ranthambore property page (Layer 2) under Portfolio. The current pass focuses on the parent site at `/` (the property page deep-dive lives at `/theme-3` and is out of scope here).

## Brand Rules

- **Public-facing name**: `AHPL` only. Do not expand the legal name anywhere on the site except legal/compliance pages.
- **Tagline**: `You belong here.`
- **Visual identity**: Modern luxury hospitality — premium, calm, confident, institutional, hospitality-led. Warm neutrals, deep earthy tones, refined typography, generous white space.
- **Avoid**: flashy visuals, generic SaaS aesthetics, anything that competes with the imagery.
- **Reference structure** (not visual copy): Chalet Hotels for parent-company-to-property navigation.

## Aesthetic Philosophy

**Refined Heritage Modernism.** Editorial magazine spreads, hairline gold rules, italic display serif accents, large amounts of negative space, photo-led storytelling. Type is the primary visual instrument.

## Typography

- Display: **Cormorant Garamond** (light/regular weights, italics for accents)
- Body: **DM Sans** (light/regular)
- Accent: **Playfair Display** (currently loaded but unused — could be removed)
